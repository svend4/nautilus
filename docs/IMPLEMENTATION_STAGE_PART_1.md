## 1. Общая характеристика

Nautilus Portal — единая точка входа для экосистемы svend4. Реализован как мультиадаптерный движок поиска и консенсуса над разнородными форматами знаний.

### Ключевые метрики кодовой базы

| Метрика | Значение |
|---------|----------|
| Всего строк Python | ~6 600 |
| Модулей в корне | 15 |
| Адаптеров | 13 |
| Тест-файлов | 4 |
| Строк тестов | ~415 |
| Внешних зависимостей | 0 (только stdlib) |
| Mypy ошибок | 0 |
| Паспортов репозиториев | 7 |
| CI/CD пайплайнов | 4 |

### Самодостаточность

Проект использует **только стандартную библиотеку Python** — никаких pip-зависимостей в рантайме. Это даёт:
- Запуск из коробки без `pip install`
- Воспроизводимость на любой машине с Python 3.10+
- Отсутствие supply-chain рисков
- Простую упаковку в Docker

### Лицензия и распространение

- Репозиторий: `svend4/nautilus`
- GitHub Pages: `svend4.github.io/nautilus`
- Контейнер: `Dockerfile` + `docker-compose.yml`
