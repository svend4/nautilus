# Паспорт: svend4/meta

| Поле | Значение |
|------|----------|
| Репозиторий | svend4/meta |
| Формат | `.meta` — CA-правила + гексаграммы |
| Единица | CA-правило (0–255) + гексаграмма (1–64) |
| Адаптер | `adapters/meta.py` |
| Уровень совместимости | 1 — читаемый |

## Структура

256 CA-правил Вольфрама → 4 класса (I–IV) → 64 гексаграммы И-Цзин → Q6-координаты.

| CA-класс | Поведение | α-эквивалент |
|----------|-----------|-------------|
| I | Фиксированные паттерны | -3 |
| II | Периодические | -1 |
| III | Хаотические | +1 |
| IV | Сложные (на грани хаоса) | +3 |

## Формула Q6

`hex_id - 1 → bin(6)` — номер гексаграммы в 6-битное представление.
